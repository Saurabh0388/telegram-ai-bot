# ClientAssist AI

## Project Overview
ClientAssist AI is a Python-based AI chatbot that automatically responds to customer queries on Telegram. It helps businesses manage inquiries about product prices, stock availability, and orders with human-like AI responses. The bot tracks conversation context, saves leads, and runs 24×7 on the cloud, ensuring timely customer communication.

## Features
- Responds automatically to customer queries on Telegram
- Handles product prices, stock, and orders
- Human-like AI conversation using OpenRouter GPT model
- Context-based multi-step conversation handling
- Stores customer leads for follow-up
- Cloud-deployed to run 24×7
- Secure API key management via environment variables

## Technology Stack
- **Programming Language:** Python
- **Messaging Platform:** Telegram Bot API
- **AI Integration:** OpenRouter (GPT-based model)
- **Libraries:** python-telegram-bot, openai
- **Cloud Platform:** Railway
- **Version Control:** Git & GitHub
